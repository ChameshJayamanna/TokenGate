# TokenGate

A simple full-stack authentication system built with **Spring Boot** and **React (Vite)**.

### Features
- User signup and login
- Email verification
- JWT-based authentication
- Protected routes

### Tech Stack
- Backend: Spring Boot, Spring Security, JWT
- Frontend: React, Vite, Axios

Notes
Sensitive values (email password, JWT secret) are not committed

Build files (target/, .jar) are excluded

More features (reset password, refresh token) will be added later
